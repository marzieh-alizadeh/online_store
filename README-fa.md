# فروشگاه اینترنتی (Online Store)

این پروژه یک نرم‌افزار فروشگاه آنلاین کوچک و کاربردی است که با زبان **پایتون (Python)** توسعه داده شده است. این سیستم دارای رابط کاربری گرافیکی (UI) بوده و از فایل‌های CSV به عنوان پایگاه داده (برای ذخیره اطلاعات کاربران، کالاها، سفارشات و نظرات) استفاده می‌کند. معماری این پروژه به صورت سرویس‌گرا (Service-Oriented) طراحی شده است تا توسعه و نگهداری آن آسان باشد.

---

## امکانات سیستم

این نرم‌افزار دارای دو نقش کاربری اصلی است که هر کدام به امکانات خاصی دسترسی دارند:

### امکانات مشتری (Customer)
* **ثبت‌نام و ورود:** ساخت حساب کاربری جدید (رمزهای عبور با الگوریتم SHA-256 به صورت امن هش می‌شوند).
* **مشاهده و جستجوی کالا:** مرور لیست محصولات فروشگاه و جستجو برای یافتن کالای مورد نظر.
* **مدیریت سبد خرید (Cart):**
  * افزودن کالا به سبد خرید.
  * مشاهده سبد خرید و محاسبه هزینه کل.
  * کم کردن تعداد یک آیتم یا حذف کامل آن از سبد خرید.
  * خالی کردن کامل سبد خرید.
* **کیف پول و خرید:** امکان افزایش موجودی حساب (Balance) و ثبت نهایی خرید و کسر مبلغ از کیف پول.
* **سیستم نظرات (Comments):** امکان نوشتن نظر برای هر کالا و مشاهده نظرات و تاریخ ثبت کامنت‌های سایر کاربران.

### امکانات مدیر فروشگاه (Admin)
* **مدیریت کالاها:** افزودن کالای جدید، ویرایش اطلاعات کالاهای موجود و مدیریت موجودی (Stock).
* **سیستم تخفیف:** امکان اعمال تخفیف روی کالاها.
* **مدیریت سفارشات:** مشاهده لیست تمامی سفارش‌های ثبت شده توسط مشتریان.
* **گزارش‌گیری:** مشاهده گزارش فروش بر اساس کالاهای مختلف.

## تکنولوژی‌ها و معماری
* **زبان برنامه‌نویسی:** Python 3.x
* **ذخیره‌سازی داده‌ها:** فایل‌های متنی `CSV` (بدون نیاز به نصب دیتابیس‌های جانبی مانند MySQL).
* **امنیت:** استفاده از ماژول `hashlib` برای رمزنگاری ایمن رمزهای عبور.
* **رابط کاربری:** دارای فرم‌های مجزا برای بخش‌های مختلف مانند ورود، پنل ادمین، فروشگاه و سبد خرید.
* **معماری:** جداسازی منطق برنامه به لایه‌های `models`، `services`، `ui` و `data`.

## ساختار پوشه‌ها و فایل‌های پروژه

\`\`\`text
online_store/
│
├── data/                  # پوشه فایل‌های پایگاه داده
│   ├── users.csv          # اطلاعات کاربران و مدیران
│   ├── products.csv       # لیست و مشخصات کالاها
│   ├── orders.csv         # تاریخچه سفارشات
│   └── comments.csv       # نظرات کاربران برای محصولات
│
├── models/                # کلاس‌های مدل داده
│   ├── user.py            # کلاس والد User و کلاس‌های Customer و Admin
│   └── product.py         # کلاس Product
│
├── services/              # منطق اصلی و پردازش‌های برنامه
│   ├── auth_service.py    # سرویس احراز هویت (لاگین/ثبت‌نام)
│   ├── product_service.py # سرویس مدیریت محصولات
│   ├── cart_service.py    # سرویس مدیریت سبد خرید (In-memory)
│   ├── order_service.py   # سرویس ثبت و مدیریت سفارشات
│   ├── comment_service.py # سرویس ثبت و خواندن نظرات
│   └── file_manager.py    # کلاس ابزار مدیریت خواندن و نوشتن فایل‌های CSV
│
├── ui/                    # فایل‌های مربوط به رابط کاربری
│   ├── login_window.py    # پنجره ورود و ثبت‌نام
│   ├── store_window.py    # پنجره اصلی فروشگاه (برای مشتری)
│   ├── admin_panel.py     # پنل مدیریت
│   ├── cart_window.py     # پنجره سبد خرید
│   ├── style.py           # استایل‌ها و تنظیمات ظاهری
│   └── utils.py           # توابع کمکی رابط کاربری
│
└── main.py                # نقطه شروع (Entry point) اجرای برنامه
\`\`\`

## راهنمای نصب و اجرا

۱. ابتدا مطمئن شوید که [Python](https://www.python.org/downloads/) نسخه 3.7 یا بالاتر روی سیستم شما نصب است.

۲. مخزن پروژه را دانلود یا Clone کنید:
\`\`\`bash
git clone https://github.com/yourusername/online_store.git
cd online_store
\`\`\`

۳. برنامه را اجرا کنید:
\`\`\`bash
python main.py
\`\`\`
*(نکته: در اولین اجرای برنامه، پوشه `data` و فایل‌های `csv` به طور خودکار توسط کلاس `FileManager` در صورت عدم وجود ساخته خواهند شد).*
